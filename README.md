# Climate_novelty
Using hypervolumes to measure climate novelty, as created in Smith et al 2022, https://journals.plos.org/climate/article?id=10.1371/journal.pclm.0000022. 

Full data is not included here, but a small working demo is provided.

For full analysis, start with the config file. For demo run the demo file.

This code is a refactored version of my original code (2021), refactored using Claude Code and Sonnet 4.6. Contact me about bugs: james.a.smith@dpi.nsw.gov.au
